# Today

> 世界上最不正经的日历应用 powered by **Kotlin**

## What's Today?

Today是一个老黄历订阅和分享平台，你可以在这里找到各种各样有趣的老黄历，也许它可以为你每天的生活指点迷津。

你也可以在这里定制专属你自己的老黄历，给自己的生活添加一些不一样的色彩~
